A custom Elementor addon that provides additional widgets and functionality for the Pedro theme.

== Description ==

Pedro Elementor Addon is a lightweight and powerful plugin that adds custom Elementor widgets and features designed specifically for the [Pedro](https://wordpress.org/themes/pedro/) theme.  
Enhance your page-building experience with pre-designed widgets and styling options built to perfectly match the Pedro theme’s design system.

== Features ==

* Custom Elementor widgets built for the Pedro theme.  
* Lightweight and optimized for performance.  
* Seamless integration with Elementor and WordPress Customizer.

== Installation ==

*** From your WordPress dashboard ***

1. Visit 'Plugins > Add New'.
2. Search for 'Pedro Elementor Addon' and install the plugin.
3. Activate 'Pedro Elementor Addon' from your Plugins page.
4. Make sure Elementor is installed and activated.

== Frequently Asked Questions ==

= Does this plugin work with any WordPress theme? =

This addon is built specifically for the [Pedro](https://wordpress.org/themes/pedro/) theme and may not work as intended with other themes.

= Do I need Elementor Pro? =

No, this addon works with the free version of Elementor. However, it’s fully compatible with Elementor Pro as well.

== Notes ==

This plugin uses Elementor’s API to register and render custom widgets.  
No external dependencies or frameworks are required.
